# Kłosie Zwiedzają

Statyczny szkic bloga podróżniczego gotowy pod GitHub Pages.

## Jak opublikować na GitHub Pages

1. Wrzuć pliki do publicznego repozytorium na GitHubie.
2. Wejdź w `Settings` -> `Pages`.
3. W sekcji `Build and deployment` wybierz `Deploy from a branch`.
4. Ustaw branch `main` i folder `/root`.
5. Zapisz ustawienia.

Po chwili strona będzie dostępna pod adresem podanym przez GitHub Pages.

## Edycja wpisów

Na razie wpisy są zdefiniowane bezpośrednio w `index.html`. To najprostszy start.
Kolejnym krokiem może być przeniesienie ich do osobnych plików Markdown i dodanie
generatora statycznego, np. Astro albo Eleventy.
